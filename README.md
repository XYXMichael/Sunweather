# SunWeather #
***
## 功能 ##
+ 调用和风天气API进行地图查找 
+ 调用和风天气API进行天气信息渲染
+ 调用百度地图API可实现定位和查找指定的地理位置
+ 实现消息通知、短信分享、华氏度转换功能
+ 使用Room数据库进行存储。不用联网也能读取存储信息
+ 实现平板适配
